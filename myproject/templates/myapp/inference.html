Вывести HTML документ из django

Создание приложения:
Нужно создать приложение, которое и будет выводить наш файл: django-admin startapp название приложения. Приложения нужны для разделения сайта на логические части, вроде чата/новостей/групп. В папке приложения создать файл urls.py, в который скопировать, содержимое такого же файла из папки проекта.

Создание директории для .html файлов:
в settings.py импортировать модуль os

-import os

Он нужен для того, чтобы мы могли изпользовать метод os.path.join() и добавить в 'DIRS' внутри списка 'TEMPLATES' директорию, прописав её относительно BASE DIR:

TEMPLATES = [
{
'BACKEND': 'django.template.backends.django.DjangoTemplates',
'DIRS': [
os.path.join(BASE_DIR, 'templates')
],
'APP_DIRS': True,
'OPTIONS': {
'context_processors': [
'django.template.context_processors.debug',
'django.template.context_processors.request',
'django.contrib.auth.context_processors.auth',
'django.contrib.messages.context_processors.messages',
],
},
},
]

Создать папку templates в указанной выше директории (как правило - рядом с файлом manage.py) в этой папке будут храниться все .html темплейты/страницы. Можно создать там .html документ, который будет выводиться

-urlpatterns = [
