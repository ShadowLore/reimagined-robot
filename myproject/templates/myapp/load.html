Установка django и запуск сервера

Если нет python:
https://www.python.org/downloads/windows/. Выбираем версию около 3.8

Создаём проект в pycharm
Слева сверху - file -> New project. Выбираем опцию с созданием виртуального окружения

Если нет django:

-pip install django

Далее прописываем в терминале Pycharm/консоли:

-django-admin startproject название проекта
-cd название проекта cd - переход в директорию(папку)
-python manage.py runserver - запуск тестового сервера

В консоли появится ip-адрес локального тестового сервера, который поднимет Django, зайти на сайт можно либо тыкнув по нему, либо введя его в поле для URL браузера
